# 🧮 iCalculator — iOS Inspired Web Calculator 🍎

A sleek, **Apple iOS–inspired calculator** built using **vanilla HTML, CSS, and JavaScript**, with a strong focus on **UI/UX polish**, **visual hierarchy**, and **Apple-style design principles**.

This project goes beyond basic functionality to deliver a calm, premium, and distraction-free user experience ✨

---

## 🚀 Features

- 🌗 **Dark & Light Modes**  
  Carefully tuned gray tones (no harsh black/white) for eye comfort

- 🍎 **iOS-Inspired UI**  
  Circular buttons, Apple-like spacing, typography, and color palette

- ⏰ **Live Time & Date Panel**  
  iOS lock-screen–style time display for contextual elegance

- 🎨 **Layered UI Design**  
  Clear separation between page background, calculator body, and buttons

- 📐 **Pixel-Perfect Alignment**  
  Centered digits, consistent spacing, and refined padding

- 🖱️ **Smooth Button Interactions**  
  Subtle press animations inspired by iOS behavior

- 🧠 **Clean Logic**  
  Simple, readable JavaScript with proper error handling

---

## 🛠️ Tech Stack

- **HTML5** — semantic structure  
- **CSS3** — Flexbox, Grid, variables, gradients  
- **JavaScript (ES6)** — calculator logic & live time updates  

No frameworks. No libraries. Just clean fundamentals 💪

---

## 🎯 Design Philosophy

This project follows **Apple Human Interface–style principles**:

- ✨ Subtlety over flashiness  
- 🎯 Focus on content, not decoration  
- 🎨 Soft contrast & neutral tones  
- 📱 UI that feels calm and intentional  

Every visual choice was refined step by step.
